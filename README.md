# React Todo App

Todo app percobaan kedua kalinya. Teknologi yang digunakan adalah React w/ vite, itu saja.

## Fitur

- Toggle dark mode.

- Modal.

- Add, edit dan remove item.

## External packages/modules

- `react-modal` digunakan untuk membuat modal edit task.
